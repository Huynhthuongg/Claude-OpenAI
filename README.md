# Claude-OpenAI (Starter)

Quick start (demo):
1. Install dependencies:
   npm install express axios
   # optional for OpenAI: npm i openai
2. Run backend:
   node server.js
3. Open browser: http://localhost:3000

Notes:
- This is a starter scaffold. Replace memory.json with Postgres + pgvector for long-term memory.
- Add your PUBLIC key to GitHub and push files. Secure skills before allowing execution.
